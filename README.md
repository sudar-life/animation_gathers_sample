# 플러터 애니메이션 정리

애니메이션을 앱에 적용하는 이유

- 사용자 경험 향상
- 시각적 매력
- 정보의 흐름 제공
- 사용자 주의 집중
- 브랜딩과 일관성

> 이 리포지토리에서는 플러터에서 사용하는 위젯들을 정리해보고 손쉽게 이해 할 수 있도록 정리하였습니다.
